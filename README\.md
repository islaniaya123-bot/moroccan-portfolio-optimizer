# 📈 Moroccan Portfolio Optimizer

A sophisticated Streamlit web application for portfolio optimization using the Moroccan financial market. This app implements the multi-period mean-variance optimization algorithm from Prince (2007) with extensions for BEKK GARCH and NGarch+Copula models.

**Perfect for:** Students, professors, and financial professionals studying portfolio theory and Moroccan market dynamics.

## ✨ Features

### 🎯 Optimization Methods
- **BEKK GARCH**: Fast real-time optimization (0.2s)
- **NGarch + Copulas**: Advanced simulation-based optimization (40s)

### 📈 Risk Measures
- **Variance (Markowitz)**: Classical mean-variance optimization
- **CVaR (Conditional Value at Risk)**: Tail risk measurement
- **Worst-Case CVaR**: Robust optimization approach

### 📊 Asset Management
Three flexible input methods:
1. **Moroccan Market Data**: Pre-loaded assets from Morocco Stock Exchange
2. **Custom Tickers**: Import any global stock tickers
3. **Manual Entry**: Enter expected returns and volatility directly

### 🔬 Analysis Features
- Historical statistics (returns, volatility, Sharpe ratio)
- Correlation matrix with heatmap visualization
- Efficient frontier visualization
- Portfolio weight allocation charts
- Mathematical methodology documentation

## 🚀 Quick Start

### Local Installation

```bash
# Clone the repository
git clone https://github.com/islaniaya123-bot/moroccan-portfolio-optimizer.git
cd moroccan-portfolio-optimizer

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

The app will open at `http://localhost:8501`

### Deploy to Streamlit Cloud

1. Push this repository to GitHub (public)
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click "New app"
4. Select this repository
5. Set main file to `app.py`
6. Click "Deploy"

## 📖 Mathematical Framework

### Mean-Variance Optimization (Markowitz 1952)
$$\min_{\mathbf{w}} \mathbf{w}^T \Sigma \mathbf{w} \quad \text{s.t.} \quad \mathbf{w}^T \boldsymbol{\mu} = R_{\text{target}}, \quad \sum_i w_i = 1$$

### Multi-Period Extension (Prince 2007)
$$\tau_t = \boldsymbol{\mu}_t^T \mathbf{V}_t^{-1} \boldsymbol{\mu}_t$$
